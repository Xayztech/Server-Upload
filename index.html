📝 Tutorial Penggunaan

Selamat datang di Server Upload Assistant! Alat ini membantumu mengunggah file ke Pixhost.to dengan mudah.

Langkah-langkah:

  1. Pilih File: Klik atau seret file ke dalam kotak putus-putus di tengah layar.
  2. Jenis File yang Didukung: Kamu bisa mengunggah gambar (JPG, PNG, GIF, BMP). Layanan ini tidak mendukung video.
  3. Ukuran Maksimal: Pastikan ukuran file tidak lebih dari 10 MB.
  4. Proses Unggah: Setelah memilih file, proses unggah akan dimulai secara otomatis. Mohon tunggu sejenak.
  5. Dapatkan Link: Jika berhasil, sebuah link akan muncul di bawah kotak unggah. Klik tombol "Salin Link" untuk menyalinnya.
  6. Jika Gagal: Jika ada masalah (misalnya file terlalu besar atau format tidak didukung), sebuah pesan kesalahan akan muncul.

Sangat mudah, bukan? Selamat mencoba!
